# simplicv

C'est un projet réalisé en vue Js et Symfony pour la génération de CV en ligne sans stockage des CV en base de données.

```bash
git clone <url-du-dépôt>
```
# front 
1. Installer les dépendances
```bash
npm i 
```
2. Lancer le serveur
```bash
npm run dev
```

# back
1. Installer les dépendances
```bash
composer i
```
2. Lancer le serveur
```bash
symfony serve
```
